# Tema 2 – Fundamentos e Técnicas Avançadas

Este trabalho demonstra a aplicação dos fundamentos da
programação juntamente com técnicas mais avançadas.

O programa utiliza funções, listas, estruturas condicionais
e de repetição para realizar operações matemáticas, cadastro
de dados e cálculo de média.

Conceitos aplicados:
- Funções
- Listas
- Estruturas condicionais
- Estruturas de repetição
- Organização do código

Linguagem utilizada: Python
